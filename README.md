Here are some of the key projects included in this repository.

### 🐍 Python Applications & Games

-   **Mastermind**
    -   A modern implementation of the classic Mastermind logic game with a graphical user interface.
    -   **Features:**
        -   Three difficulty levels (Easy, Medium, Hard).
        -   Timed challenges and limited attempts.
        -   Dynamic scoring system.
        -   Online high-score leaderboard using Firebase Realtime Database.
        -   Multilingual support (French/English).
    -   **Technologies:** Python, `customtkinter`, `requests`, Firebase.

-   **Career Orientation Chatbot**
    -   A simple but effective chatbot designed to guide users toward potential career paths and relevant academic specializations based on their interests.
    -   **Technologies:** Python, `customtkinter`, `nltk`.

-   **Survivor**
    -   An immersive survival game where the player must navigate challenges and stay alive.
    -   **Technologies:** Python, Pyxel.

-   **AI & NLP Scripts**
    -   Includes experimental scripts for detecting emotions (`detect.py`) and literary devices (`procédés_detect.py`) in text using regular expressions and basic NLP techniques.
    -   **Technologies:** Python, `matplotlib`.

## ⚙️ Running the Projects

### Web Projects

The web applications (Netflix Clone, Pokédex, E-commerce Site) are hosted online. You can access them directly via the "Live Demo" links above.

### Python Projects

To run the Python applications, you need Python 3.x installed on your system.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/imkk17/portfolio.git
    cd portfolio
    ```

2.  **Mastermind:**
    -   Install the required libraries:
        ```bash
        pip install customtkinter requests
        ```
    -   Run the game:
        ```bash
        python MasterMindV6_final.py
        ```

3.  **Career Orientation Chatbot:**
    -   Install the required libraries:
        ```bash
        pip install customtkinter nltk
        ```
    -   Run the application:
        ```bash
        python projet/career_chat_bot.py
        ```
        *Note: NLTK may prompt you to download additional data on the first run.*

4.  **Survivor Game:**
    -   Unzip the `project_survivor1.zip` file.
    -   Install the required libraries:
        ```bash
        pip install pyxel
        ```
    -   Navigate into the unzipped directory and run the main file (e.g., `python menu.py`).
